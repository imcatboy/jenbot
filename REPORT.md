1 Характеристика предприятия

Производственная практика проходила в Федеральном государственном автономном научном учреждении «Научно-исследовательский институт «Спецвузавтоматика»» (ФГАНУ «НИИ «Спецвузавтоматика»).
ФГАНУ «НИИ «Спецвузавтоматика» является государственным научно-исследовательским учреждением, осуществляющим деятельность в области разработки программного обеспечения, автоматизированных систем управления, информационных технологий, защиты информации, цифровой обработки данных и научно-исследовательских работ.
Основными направлениями деятельности предприятия являются:
проведение научно-исследовательских и опытно-конструкторских работ;
разработка программного обеспечения различного назначения;
создание автоматизированных информационных систем;
разработка средств защиты информации;
внедрение современных информационных технологий;
сопровождение и модернизация программных продуктов.
Организационная структура предприятия включает руководство организации, административные подразделения, научно-исследовательские отделы, отделы разработки программного обеспечения, отдел тестирования, подразделения технического сопровождения и вспомогательные службы.
Практика проходила в подразделении, занимающемся разработкой программного обеспечения. Основными задачами отдела являются проектирование, разработка, тестирование, сопровождение и модернизация программных продуктов.
Работа отдела осуществляется поэтапно и включает анализ требований заказчика, проектирование архитектуры программного продукта, разработку программного кода, тестирование, исправление выявленных ошибок, подготовку технической документации и сопровождение разработанного программного обеспечения.
В период прохождения практики было выполнено индивидуальное задание по разработке программного средства классификации текстовых сообщений с использованием методов машинного обучения. В процессе выполнения задания были изучены процессы проектирования программного обеспечения, особенности разработки интеллектуальных систем обработки текстовой информации и современные средства разработки на языке Python.

2 Компьютерное оснащение предприятия

Рабочее место техника-программиста было оборудовано современным персональным компьютером, обеспечивающим выполнение задач по разработке, тестированию и сопровождению программного обеспечения.
Основные технические характеристики рабочего места:
процессор Intel Core i5 (или аналогичный по производительности);
оперативная память — 16 ГБ;
SSD-накопитель объёмом 512 ГБ;
монитор с разрешением Full HD;
клавиатура и оптическая мышь;
подключение к локальной вычислительной сети предприятия и сети Интернет.
Для выполнения индивидуального задания использовалось следующее программное обеспечение.
Операционная система Windows 11 Pro — обеспечение работы программных средств, управление аппаратными ресурсами компьютера.
Visual Studio Code — среда разработки, использовавшаяся для написания, редактирования и отладки программного кода.
Python 3.11 — язык программирования, применявшийся при разработке программного средства классификации текстовых сообщений.
pandas — библиотека для загрузки, обработки и анализа набора текстовых данных.
scikit-learn — библиотека машинного обучения, использовавшаяся для подготовки данных, обучения модели классификации, тестирования и оценки качества модели.
CustomTkinter — библиотека для разработки современного графического интерфейса пользователя.
matplotlib — библиотека для построения графиков, анализа данных и визуализации результатов обучения модели.
joblib — библиотека для сохранения обученной модели машинного обучения и последующей загрузки при работе программы.
Использование данного аппаратного и программного обеспечения позволило выполнить полный цикл разработки программного средства: анализ данных, обучение модели машинного обучения, тестирование, визуализацию результатов, разработку графического интерфейса и подготовку технической документации.

3 Должностная инструкция техника/техника-программиста

Техник-программист относится к категории специалистов и назначается приказом руководителя организации.
Основными обязанностями являются:
разработка программного обеспечения;
анализ требований;
написание программного кода;
тестирование;
сопровождение программ;
исправление ошибок;
разработка технической документации;
участие в модернизации существующих программных решений;
соблюдение требований информационной безопасности.
Работник имеет право:
получать необходимую техническую документацию;
пользоваться программными средствами предприятия;
вносить предложения по совершенствованию программных продуктов;
получать консультации специалистов.
Работник несёт ответственность за:
качество разработанного программного обеспечения;
соблюдение сроков выполнения работ;
сохранность служебной информации;
соблюдение требований техники безопасности.

4 Техника безопасности при работе на ПК

Перед началом работы необходимо проверить исправность оборудования, отсутствие повреждений кабелей питания и периферийных устройств.
Монитор должен находиться на расстоянии 60–70 см от пользователя, верхняя граница экрана — немного ниже уровня глаз.
Рабочее место должно иметь достаточное освещение без появления бликов на экране.
Каждые 45–60 минут рекомендуется делать перерыв продолжительностью 5–10 минут.
Запрещается:
работать с неисправным оборудованием;
самостоятельно ремонтировать технику;
прикасаться к электрическим соединениям влажными руками;
загромождать вентиляционные отверстия системного блока.
При возникновении неисправности необходимо отключить оборудование и сообщить руководителю.
